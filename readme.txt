〇main.m
・分析したいファイルのパスを変更
・mainは大きく、modified.m・Export_fig.m・Lyapunov.mの三つの関数を実行
　・その他の関数は、それぞれのファイルに必要な関数
・funcにmain.m以外のファイルを保存しており、ここを参照することで実行できるようにしている

〇subroutine
・modified.mだけ実行したかったり、Export_fig.mだけ実行したかったりする際に使用するファイル
・データの前処理は同じだが、グラフだけ変更したいときなどは、Export_fig.mの内容を変更して実行することで不必要な処理を省略
